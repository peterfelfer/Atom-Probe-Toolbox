\n\n\n \n calculate2Dprofile - Atom Probe Toolbox \n\n

calculate2Dprofile

\n calculates 2D concentration profile of 'atoms' compared to 'allAtoms'. If
 'allAtoms' is not parsed, raw counts will be used. 'crop' is a [min max]
 limit for the 2D profile. The 'normalAxis' also defines the origin of the
 2D concentration profile.
