patchToStl

  patchToStl exports a patch (FV) structure to an STL file.

  patchToStl();
            opens a "Save *.stl file to" window if selected object is a patch
  patchToStl(fv);
            opens a "Save *.stl file to" window
  patchToStl(fv, fileName);
            saves file in current folder or specified path
  patchToStl(fv, fileName, 'format', 'binary');
            saves binary STL

  INPUTS:
  fv:        structure with faces (f) and vertices (v)
  fileName:  desired name of saved file with .stl as suffix
