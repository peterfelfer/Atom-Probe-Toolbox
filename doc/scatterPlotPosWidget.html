scatterPlotPosWidget

SCATTERPLOTPOSWIDGET Interactive scatter plot with visibility/sample controls.

[scatterHandles, ax, controlFig, info] = scatterPlotPosWidget(pos, colorScheme)
[scatterHandles, ax, controlFig, info] = scatterPlotPosWidget(pos, colorScheme, 'groupBy', 'ion')
[scatterHandles, ax, controlFig, info] = scatterPlotPosWidget(pos, [], 'axes', ax)

Creates a 3D scatter plot of APT data and a separate control window that
toggles species visibility, sampling fraction, and marker-size settings.
State is persisted on the target axes (ax.UserData.scatterPlotPosWidgetState)
and restored automatically when reopening on the same axes.

OPTIONS (selection):
  'axes'                 - target axes for scatter plot
  'sample'               - initial sample fraction (0..1) or absolute count (>1)
  'markerSize'           - default marker size
  'markerSizeGlobal'     - global marker size applied to linked species
  'restoreStateFromAxis' - restore saved state from target axes (default: true)
  'persistStateToAxis'   - persist state to target axes during interaction

STATE WORKFLOW:
  % Open widget and persist state on axes automatically
  [~, ax, controlFig] = scatterPlotPosWidget(pos, colorScheme);

  % Save explicit state in workspace/file
  state = scatterPlotPosWidgetGetState(controlFig);
  save('scatterState.mat', 'state');

  % Restore explicit state to a running widget
  load('scatterState.mat', 'state');
  scatterPlotPosWidgetApplyState(controlFig, state);

  % Re-open widget from a saved visualization/figure axes
  fig = openfig('savedScatter.fig');
  ax = findobj(fig, 'Type', 'axes');
  scatterPlotPosWidget(pos, colorScheme, 'axes', ax(1));
